---
title: Installation
description: Install Minions Risks packages
---

## TypeScript / Node.js

```bash
npm install @minions-risks/sdk minions-sdk
```

## Python

```bash
pip install minions-risks
```

## CLI

```bash
npm install -g @minions-risks/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
