# Minions Risks

Risk register, likelihood, impact, and mitigation plans

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-risks/sdk`) — Core library for Node.js
- **Python SDK** (`minions-risks`) — Python bindings
- **CLI Tool** (`@minions-risks/cli`) — Command-line interface
- **Web App** — Interactive playground at [risks.minions.wtf](https://risks.minions.wtf)
