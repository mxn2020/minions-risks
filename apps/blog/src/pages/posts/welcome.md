# Welcome to Minions Risks

We're excited to introduce **Minions Risks** — Risk register, likelihood, impact, and mitigation plans.

## What's Included

- **TypeScript SDK**: `npm install @minions-risks/sdk`
- **Python SDK**: `pip install minions-risks`
- **CLI**: `npm install -g @minions-risks/cli`

## Getting Started

Check out the [documentation](https://risks.minions.help) for a full guide.
